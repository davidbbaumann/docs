## Overview

Use the Content Management Agent (CMA) in Nexus Search to **quickly and easily generate automated image descriptions, SEO-relevant alternative texts, and ARIA descriptions for accessibility requirements**. No knowledge of search engine optimization or image optimization is required. The Content Management Agent handles this for you.

## Capabilities of the Content Management Agent (CMA)

### Internal Image Descriptions

A longer description for the Content Studio library, always generated by the agent. It describes the subject of the image/creative, the visual style, and suitable use cases such as channel suitability (e.g., LinkedIn header, StepStone banner). The description may include editorial notes.

### Alt Texts / Alternative Texts

A concise, functional description of the image with a maximum of 125 characters. These serve three key functions:

- Screen readers use the text to describe the image and make its content understandable.
- If an image is blocked in a browser or cannot be loaded, the text appears in place of the image.
- Search engines use the text to capture image content and context.

### ARIA Descriptions (aria-describedby)

A detailed, longer description in accordance with WCAG ARIA15 for use with aria-describedby. For complex images only. 2–5 complete sentences are generated describing the creative/image. Any text embedded within the image must be included verbatim in the description.

## Generating the Descriptions

1. Open **Settings \> Content Studio** in the left sidebar navigation.
2. Open the **"Images" tab**.
3. Click the **"Upload image" button**.
4. Enable **"Have image analyzed by the Content Management Agent"**. This is enabled by default.
5. The detail view of the image/creative opens and you can adjust the descriptions as needed.

## Editing & Deleting Descriptions

The CMA only generates texts if it was enabled during the upload process. It is possible at any time to edit an image/creative and make changes to the descriptions. Descriptions can also be deleted. The only requirement is that a name for the image must be present.
