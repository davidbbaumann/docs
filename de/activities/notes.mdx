## Übersicht

Notizen sind flexible Datensätze in Ihrer Aktivitäten-Zeitleiste, die Ihnen helfen, wichtige Informationen, Beobachtungen und Erkenntnisse über Ihre [Unternehmen](/de/core-concepts/accounts), [Kontakte](/de/core-concepts/contacts), [Kandidaten](/de/core-concepts/candidates), [Projekte ](/de/core-concepts/projects)und [Bewerbungen ](/de/core-concepts/applications)zu erfassen. Im Gegensatz zu anderen Aktivitätstypen, die spezifische Interaktionen verfolgen (Anrufe, Termine, E-Mails), dienen Notizen als Ihr digitaler Notizblock zur Dokumentation von allem, was für Ihre Arbeit wichtig ist.

## Eigenschaften von Notizen

| Feld                      | Beschreibung                                                                                                                                            | |
| :------------------------ | :------------------------------------------------------------------------------------------------------------------------------------------------------ | :-- |
| Verbundene Datensätze     | Datensätze in Nexus, die mit der Notiz verbunden sind. Wenn ein Datensatz verbunden ist, taucht die Notiz in der Aktivitätenleiste des Datensatzes auf. | |
| Vorlage verwenden         | Wählen Sie eine Notiz-Vorlage aus dem Content Studio. Es werden die definierten Inhalt inkl. der Variablen geladen.                                     | |
| Notiztitel                | Titel der Notiz                                                                                                                                         | |
| Notiztyp                  | Art der Notiz. Kann folgende Werte haben: **Allgemeine Notiz**, **Meeting-Notiz**, **Anrufnotiz**, **Recherche**, **Erinnerung**                        | |
| Notizinhalt               | Optionale Beschreibung der Notiz                                                                                                                        | |
| Diese Notiz anheften      | Wenn die Notiz angeheftet wird, wird sie immer ganz oben in der Aktivtätenleiste der verbundnen Datensätze angezeigt.                                   | |
| Diese Notiz privat machen | Wenn die Notiz privat ist, kann nur der Ersteller die Notiz sehen.                                                                                      | |

## Notiz erstellen

<Steps>
  <Step title="Öffnen Sie die Detailseite des Datensatzes (Unternehmen, Kontakt, etc.) an dem Sie eine Notiz erstellen möchten">
    
  </Step>
  <Step title='Öffen Sie den Tab "Aktivitäten"'>
    
  </Step>
  <Step title='Klicken Sie in der oberen rechten Ecke der Aktivitätenleiste auf den Button "Notiz"'>
    Folgende Felder sind bearbeitbar:

    - Verknüpfte Datensätze
    - Vorlage verwenden
    - Notiztitel
    - Notiztyp
    - Notizinhalt
    - Diese Notiz anheften
    - Fälligkeitsdatum
    - Diese Notiz privat machen
  </Step>
  <Step title='Klicken Sie auf den blauen Button "Notiz erstellen" zur Speicherung'>
    Klicken Sie auf Abbrechen, wenn Sie die Notiz nicht erstellen möchten.
  </Step>
</Steps>
