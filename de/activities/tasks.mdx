## Übersicht

Aufgaben helfen Ihnen, Ihre To-Do-Listen in Nexus Search zu organisieren und zu verfolgen. Sie können mit verschiedenen Datensätzen, wie Unternhemen, Kontakten, Projekte oder Kandidaten verbunden werden und sind Teil der Aktivitäten-Zeitleiste der verbundenen Datensätze.

## Eigenschaften von Aufgaben

| Feld                  | Beschreibung                                                                                                                                               | |
| --------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------- | --- |
| Verbundene Datensätze | Datensätze in Nexus, die mit der Aufgabe verbunden sind. Wenn ein Datensatz verbunden ist, taucht ie Aufgabe in der Aktivitätenleiste des Datensatzes auf. | |
| Vorlage wählen        | Wählen Sie eine Aufgaben-Vorlage aus dem Content Studio. Es werden die definierten Inhalt inkl. der Variablen geladen.                                     | |
| Titel                 | Titel der Aufgabe **(Pflicht)**                                                                                                                            | |
| Beschreibung          | Optionale Beschreibung der Aufgabe                                                                                                                         | |
| Priorität             | Gibt die Wichtigkeit der Aufgabe an. Werte sind: **Niedrig**, **Mittel** oder **Hoch**                                                                     | |
| Status                | Gibt den aktuellen Stand der Aufgabe an. Werte sind: **Nicht begonnen**, **In Bearbeitung**, **Abgeschlossen**, **Abgebrochen**                            | |
| Startdatum            | Das Datum, an dem mit der Aufgabe begonnen werden soll.                                                                                                    | |
| Fälligkeitsdatum      | Das Datum, an dem die Aufgabe abgeschlossen werden soll.                                                                                                   | |
| Geschätzte Stunden    | Gibt an, wie viel Zeit für die Aufgabe eingeplant werden muss                                                                                              | |
